/* ============================================================ */
/* CRAFTING NOTES                                                */
/* ============================================================ */

/* All journal crafting is done through the standard crafting    */
/* menu (B key). Available recipes:                              */
/*                                                                */
/* BIND (Journal/Notebook -> Survival Journal):                  */
/*   - Thread + Any Cover (3x binding material + thread + needle)*/
/*   - Adhesive + Any Cover (10x binding material + tape/glue)   */
/*                                                                */
/* RESTORE (Worn/Bloody -> Clean Filled):                        */
/*   - Thread + Any Cover (1x binding + thread + needle)         */
/*   - Adhesive + Any Cover (5x binding + tape/glue)             */
/*                                                                */
/* Thread recipes require Tailoring 1 and award 3 Tailoring XP.  */
/* Adhesive recipes have no skill requirement but use more mats. */
