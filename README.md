# Burd's Survival Journals

A Project Zomboid mod for Build 42 that lets you record your survival skills in personal journals and learn from fallen survivors.

## Features

- **Craft Survival Journals** - Bind blank journals using leather strips and thread
- **Record Your Skills** - Write your current skill levels into journals for other characters to learn from
- **Find Worn Journals** - Discover journals in bookshelves, desks, and other containers throughout the world
- **Loot Bloody Journals** - Search zombie corpses for journals from fallen survivors with rare XP and trait rewards
- **Restore Damaged Journals** - Clean bloody journals to recover the knowledge within
- **Multiplayer Support** - Share journals with other players (configurable via sandbox options)

## Installation

### Steam Workshop
Subscribe to the mod on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3394230024)

### Manual Installation
1. Download or clone this repository
2. Copy the `Contents/mods/BurdSurvivalJournals` folder to your Project Zomboid mods directory:
   - Windows: `%UserProfile%\Zomboid\mods\`
   - Linux: `~/Zomboid/mods/`
   - Mac: `~/Zomboid/mods/`

## Compatibility

- **Build 42** - Full support (uses `42/` folder)
- **Build 41** - Legacy support (uses `common/` folder)

## Sandbox Options

- **Allow Others to Open Journals** - Let other players read your journals in multiplayer
- **Allow Others to Claim from Journals** - Let other players gain XP from your journals

## Crafting Recipes

| Recipe | Ingredients | Skill Required |
|--------|-------------|----------------|
| Bind Journal as Survival Journal | Journal + 3 Leather Strips + Thread + Needle | Tailoring 1 |
| Restore Bloody Journal | Bloody Journal + Cloth/Rag + Water | None |
| Restore Worn Journal | Worn Journal + Glue | None |
